h2. <a name="v0.9.8">v0.9.8</a>

* Break changes
** Automatic escape expression using escapeHtml if the template file name suffix is .html
** "app/view" as template root are no longer supported. All rythm template files must be put into "app/rythm" folder
** "app/view/tags/rythm" as tag root are no longer supported. Just put tag files as normal template file into "app/rythm" folder
** <code>@section</code> tag usage:
*** previously <code>@section sectionName {...}</code>
*** now <code>@section("sectionName") {...}</code>
* New features
** Add "@raw() {...}":reference#raw keyword
** Add "@escape(...) {...}":reference#escape keyword
** Add "@cache(...)":reference#cache keyword
** "Cache tag invocation result":user_guide#tag_invocation_cache
** <a href="reference#for_second_form">Support <code>@for(int i = 0; i < 100; ++i)</code> loop style</a>
** <a href="reference#return">Abort current template executing process and return to caller using <code>@return()</code></a>
** <a href="reference#break">Break the current loop: <code>@for(...) {... @break() ...}</code> using <code>@break()</code>"</a>
* Small enhancements
** <code>@verbatim(){}</code> will escape html code
** <code>@renderBody()</code> now works inside a tag and output the passed in tag body, replacing <code>@_body</code> variable
** "Invoke tags using relative path and import path":user_guide#call_tag_relative_path
** Clear alias for Play specific keywords:
*** <code>@url</code>: <code>@_u</code>, url reverse lookup
*** <code>@fullUrl</code>: <code>@_au</code>, url reverse lookup with absolute path
*** <code>@msg</code>: <code>@_m</code>, i18n message lookup
*** <code>@render() without parameter</code> now equals to <code>@renderBody()</code>
